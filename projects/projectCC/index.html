Automatic Sync Phone to Cloud

In this personal project, I aimed to reduce monthly expenses on Google storage by synchronizing my phone's gallery with AWS S3. I created an HTML webpage to display the synchronized media, using AWS services including S3, Lambda, and CloudFront for cost-effective storage, server-side processing, and low-latency content delivery. This project offers an economical alternative for personal media storage while harnessing the power of AWS's scalable cloud services.
